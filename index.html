3: Button

Now we'll listen for a button press. Connect the two wires to the GPIO pins as follows:

One wire -> GND   The other wire -> GPI25

It doesn't matter which wire goes to which pin. Like you did the past lesson, create a new file called Lesson3.py and to the next slide to see the code.

3: Button Press


from gpiozero import *
button = Button(25)

def turn_on():
    print("Hi!")

def turn_off():
    print("Bye!")

while True:
    button.when_pressed = turn_on
    button.when_released = turn_off

It'll say "Hi!" when pressed and "Bye!" when released. If not, time to get good at debugging! Did you know the best coders are the best dubuggers?!

3: Button Review

  1. infinite loops with while True; instead of just looping a few times.
  2. creating a function with def turn_off(): and def turn_on():
  3. with the new functions we created, we can have them called when pressed button.when_pressed = turn_on and released button.when_released = turn_off
  4. creating a button object using GPIO pin 25 with button = Button(25)

4: Function lightCheck()

This function checks if the light is on, checks which player pressed the button and gives you a point or takes away a point: 


def lightCheckPrintScore(player):
    player = player - 1
    if led.is_lit:
        led.off()
        players[player] += 1
    else:
        players[player] -= 1

    print()
    print("Player 1: " + str(players[0]))
    print("Player 2: " + str(players[1]))

4: Function buttonCheck()

This function simply checks if a button was pressed and then calls lightCheckPrintScore() passing it the button that was pressed. 


def buttonCheck(button_number):
    if button_number == 1:
        return lambda: lightCheckPrintScore(button_number)
    if button_number == 2:
        return lambda: lightCheckPrintScore(button_number)

4: Endless Loop

We'll need to endlessly loop and wait a random amount time here: 


while True:
    button1.when_pressed = buttonCheck(1)
    button2.when_pressed = buttonCheck(2)

    led.on()
    time.sleep(randrange(5))

from gpiozero import *
from random import randrange
import time

players = [0, 0]
button1 = Button(25)
button2 = Button(12)
led = RGBLED(18, 23, 24)


def lightCheckPrintScore(player):
    player = player - 1
    if led.is_lit:
        led.off()
        players[player] += 1
    else:
        players[player] -= 1

    print()
    print("Player 1: " + str(players[0]))
    print("Player 2: " + str(players[1]))


def buttonCheck(button_number):
    if button_number == 1:
        return lambda: lightCheckPrintScore(button_number)
    if button_number == 2:
        return lambda: lightCheckPrintScore(button_number)


while True:
