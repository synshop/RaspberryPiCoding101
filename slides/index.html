0: Prep your Raspberry Pi

  1. Assemble your case around your Raspbery Pi
  2. Insert Micro SD Card into bottom of Raspbery Pi
  3. Plug HDMI cable into monitor and Raspbery Pi (use left HDMI on Pi)
  4. Plug Keyboard and mouse into Raspbery PI
  5. Put the GPIO leaf over the GPIO pins
  6. Plug the Power supply into the powerstrip and finally into the Raspberry Pi

Your Pi should now boot up and display on the monitor

1: Hello World++

Let's add the concept of a while loop and a few variables. Edit your code to look like it does below and where it says YOUR_NAME put your name instead: 

name = "YOUR_NAME"
count = 1
while count < 6:
	print("Hi " + name + " count is " + str(count))
	count += 1
print("Bye!")

Click the button and then the button. 5 lines should show like: Hi Bob count is 4.

1: Hello World wrap up

Nice work!

You should now know how to assign variables as seen with name and count. As well, we made a while loop run for 5 times increasing the count variable by one each time. Finally, you have to start and stop the code when you make changes want to try the code again.

Feel free to go back and change this code to do different things. Can you make it run 100 times? Print emoji ü§∑‚Äç‚ôÄÔ∏è? What if you do count += 2? Have fun with it!
