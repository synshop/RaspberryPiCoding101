0: Set up your Raspberry Pi

  1. Insert Micro SD Card into bottom of Raspbery Pi
  2. Plug HDMI cable into monitor and Raspbery Pi (use left HDMI on Pi)
  3. Plug Keyboard and mouse into Raspbery PI
  4. Put the GPIO leaf over the GPIO pins
  5. Plug the Power supply into the powerstrip and finally into the Raspberry Pi

Your Pi should now boot up and display on the monitor

1: Hello World++

Let's add the concept of a while loop and a few variables. Edit your code to look like it does below and where it says YOUR_NAME put your name instead: 

name = "YOUR_NAME"
count = 1
while count < 6:
	print(" Hi " + name + " count is " + str(count))
	count += 1
print("Bye!")

Click the button and then the button. 5 lines should show like: Hello Bob count is 4.
